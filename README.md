# SMC Intraday Bot

Bot Telegram untuk sinyal trading **SMC intraday (LONG only)** yang otomatis scan semua pair USDT di Binance dan mengirim sinyal ke Telegram.


```
## Setup

### 1. Buat bot Telegram

- Chat ke **@BotFather**
- `/newbot` → ambil **BOT TOKEN**

### 2. Ambil chat ID

- Chat ke **@userinfobot**
- Catat `Your user ID` → itu **TELEGRAM_CHAT_ID**

### 3. Clone / download repo ini
```
